유용한 소프트웨어 정리

1. notepad++
텍스트 에디터
메모장이랑 다르게 저장을 안 해도 그 기록이 남아 있으며, 파일의 확장자를 지정해주면
해당 소스에 맞게 하이라이트를 적용해준다.
괄호 등의 텍스트에 대해서 시작부분과 끝부분을 하이라이트로 지정해줌

https://notepad-plus-plus.org/downloads/v7.9.5/
Download 64-bit x64 밑에 있는 Installer 받으면 됨

2. visual studio code
텍스트 에디터 
notepad++과 유사하되 조금 더 무거운 텍스트 에디터.
alt 키를 활용하여 여러 개를 선택 가능하다.
각종 확장팩들 설치에 따라서 기능들이 점점 더 강력해진다.
html 소스 작성시 신세계를 맛볼 수 있음
https://code.visualstudio.com/

유료sw이나 무료처럼 쓸 수 있는 방법들이 인터넷에 많으니 참고하면 된다.
유사한 sw로는 윈머지가 있으며 이 것은 100% 무료이다.
https://www.scootersoftware.com/download.php?zz=dl4

7. zoomIt
자유자재로 화면을 확대하고 글자를 적을 수 있음 
기본 설정은 아래와 같으나 변경 가능
ctrl+1 확대
ctrl+2 글자쓰기
ctrl+3 타이머
ctrl+4 화면 확대하되 마우스 커서랑 같이 확대
참고로 윈도우키랑 + 키를 동시에 누르면 화면이 확대되고, 여러 번 누르면 더 확대 된다.
그리고 윈도우 키랑 esc 키를 같이 누르면 확대가 풀린다. 
하지만 zoomIt이 더 쓰기 편한 것 같다.
https://m.blog.naver.com/alice8565/221959202005
